## How to run

### Prerequisites

Required Libraries:
* pytorch

### Data preparation

* Download the dataset from [here](https://drive.google.com/drive/folders/1UDfmivMeFUQf3ovLU9UFz5N8JACRtE3k?usp=sharing)
* Put all the files to `.data\movie-corpus`

### Train model and Chat with bot
```
python main.py
```
